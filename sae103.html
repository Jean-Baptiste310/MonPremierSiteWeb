SAE 1.03 - Installation d'un poste pour le développement

Présentation du projet

Nom du projet : Installation d'un poste pour le développement

Mots-clés : Machine virtuelle, Recherche, Installation, Configuration

Contexte : Deux clients nous demandent de proposer une configuration matérielle adaptée à leurs besoins ainsi que l'installation de leurs machines virtuelles.

But : Concevoir une configuration matérielle, installer et configurer les machines virtuelles nécessaires au développement.

Nombre d'étudiants : 2 étudiants

Temps passé : 25 heures

Ressources utilisées : Ubuntu, VirtualBox, Word, LDLC

Résultat final

Lors de cette SAE, j'ai réalisé la configuration matérielle destinée à une famille en sélectionnant des composants adaptés aux besoins exprimés par les clients. J'ai ensuite créé deux machines virtuelles, l'une sous Windows et l'autre sous Ubuntu.

Sur la machine virtuelle Windows, j'ai configuré les différents comptes utilisateurs ainsi que le contrôle parental destiné au plus jeune enfant. Sur la machine Ubuntu, j'ai effectué la préinstallation de plusieurs outils de développement, notamment g++, MySQL et Sublime Text.

Avec davantage de temps, j'aurais pu proposer des composants additionnels pour enrichir la configuration de la famille. Il aurait également été intéressant de développer davantage de scripts et d'automatisations afin de rendre les machines virtuelles plus complètes.

À la suite de ce projet, nous aurions pu poursuivre le développement en réalisant des scripts plus avancés ainsi qu'une simulation des performances de la configuration matérielle proposée aux clients.

Apports

Ce projet m'a permis d'approfondir mes connaissances sur les composants d'un ordinateur, notamment leur compatibilité et leur rôle dans une configuration. J'ai également renforcé mes compétences dans l'utilisation des machines virtuelles, la création de comptes utilisateurs et l'installation d'un environnement de développement. Enfin, cette SAE m'a permis de mieux comprendre les étapes de préparation d'un poste de travail destiné au développement informatique.
