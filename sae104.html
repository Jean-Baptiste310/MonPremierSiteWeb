SAE 1.04 - Création d'une base de données

Présentation du projet

Nom du projet : Création de base de données

Mots-clés : Dictionnaire, MCD/MLD, SQL, Requêtes, Base de données

Contexte : Une agence de voyage souhaite mettre en place un site web. Pour cela, elle nous demande de concevoir une base de données regroupant toutes les structures d'information nécessaires au fonctionnement de son application.

But : Réaliser une base de données comprenant un dictionnaire de données, un MCD/MLD, le code SQL ainsi que des requêtes permettant à l'entreprise d'exploiter les informations dont elle a besoin.

Nombre d'étudiants : 3 étudiants

Temps passé : 20 à 25 heures

Ressources utilisées : Looping, Word, MySQL Workbench

Résultat final

Lors de cette SAE, nous avons réalisé une base de données complète comprenant un dictionnaire de données, un MCD, un MLD, un script SQL généré à partir du modèle ainsi qu'une série de requêtes permettant de manipuler et d'exploiter les données. Nous avons également rédigé un rapport présentant les choix effectués tout au long du projet.

Le principal point fort de notre travail est la conception d'un MCD cohérent avec le cahier des charges. Le code SQL est également complet et respecte ce modèle, avec la mise en place de contraintes et de différentes requêtes d'interrogation de la base de données.

Avec davantage de temps, nous aurions pu simplifier le dictionnaire de données afin de le rendre plus clair et plus synthétique. Nous aurions également pu enrichir la base en ajoutant davantage de commandes INSERT et de requêtes SQL pour illustrer un plus grand nombre de cas d'utilisation.

À la suite de ce projet, il aurait été possible de poursuivre le développement en créant le site web de l'agence de voyage, en s'appuyant directement sur la base de données réalisée.

Apports

Ce projet m'a permis de concevoir des bases de données de plus grande ampleur, avec un nombre plus important de tables et de relations. Le travail en équipe m'a également appris à mieux répartir les tâches au sein d'un groupe de projet. Enfin, j'ai renforcé mes compétences dans la conception de MCD et de MLD, la rédaction de scripts SQL ainsi que dans l'organisation des différentes étapes nécessaires à la création d'une base de données.
